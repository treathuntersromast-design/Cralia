> 作成日: 2026-05-14  
> 背景: リリース直前の議論。AIチャットボット等のAPIコストが利用頻度に比例して発生するため、有料プランで回収する設計を検討。

## 推奨プラン構成

### Free（無料）

ユーザー獲得を妨げないよう基本機能は開放。マッチング成立が収益の源泉なので依頼フローはフリーで提供。

- 検索・プロフィール閲覧
- 依頼の作成・受注（同時進行 3件まで）
- メッセージ・通知
- エスクロー決済（手数料は別途）
- ポートフォリオ登録（5件まで）
- 営業メッセージ送信（月5通まで）
- AIクリエイター提案（月3回まで）
- レビュー・イベント閲覧

### クリエイタープロ（月額）

クリエイターは仕事が増えるほど価値が上がるため、継続課金につながりやすい。

| 機能 | 内容 |
|---|---|
| AI営業メッセージ作成・添削 | 無制限（無料: 月3回） |
| AI自己紹介文作成 | 無制限（無料: 初回1回のみ） |
| AI仕事募集文作成 | 無制限（無料: 初回1回のみ） |
| ダッシュボードチャットボット | 無制限（無料: 1日5回） |
| 同時進行の依頼数 | 無制限（無料: 3件） |
| ポートフォリオ登録 | 無制限（無料: 5件） |
| 営業メッセージ送信 | 無制限（無料: 月5通） |
| 検索上位表示 | バッジ表示・優先表示 |

### 依頼者プロ（月額または都度課金）

依頼文の質が上がるほどマッチング率が上がり、依頼者に直接的な価値がある。

| 機能 | 内容 |
|---|---|
| AI依頼文作成・添削チャット | 無制限（無料: 1日3回） |
| AIクリエイター提案 | 無制限（無料: 月3回） |
| 同時進行の依頼数 | 無制限（無料: 3件） |
| お気に入りクリエイター保存 | 無制限（無料: 10人） |

## 最初に有料化すべき機能

コストと価値の両面から **「AI依頼文チャット」と「AI営業メッセージチャット」** を最優先とする。  
理由: 会話が長くなるほどトークン消費が増え、かつユーザーが最も価値を感じる機能のため。

## 実装方針（未実装・将来対応）

### DB変更

```sql
-- users テーブルにプランカラムを追加
ALTER TABLE public.users
  ADD COLUMN plan TEXT NOT NULL DEFAULT 'free'
    CHECK (plan IN ('free', 'creator_pro', 'client_pro'));
```

### レート制限の変更

現状 `lib/aiGuard.ts` の `checkRateLimit` で日次制限を実施済み。  
有料化時はプランに応じて `limit` を切り替える、または有料ユーザーはチェックをスキップする形で対応。

```typescript
// イメージ
const limit = user.plan === 'free' ? 3 : Infinity
```

### Stripe サブスクリプション

現状のエスクロー決済は `mode: 'payment'`（単発）。  
サブスク用に `mode: 'subscription'` の別フローが必要。

- Stripe Dashboard でサブスク商品・価格を作成
- `/api/payments/create-subscription` エンドポイントを新設
- Webhook で `customer.subscription.updated` / `customer.subscription.deleted` を処理してプランを更新
